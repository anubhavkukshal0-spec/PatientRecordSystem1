# Patient Record System 

A robust Java Swing application for managing patient records and medical history. This project was developed as a Unit VI Capstone.
## Features

- **Patient Management**: Add, update, and view patient details.
- **Database Integration**: MySQL backend for persistent storage.


## Prerequisites

- Java 17 or higher
- Maven
- MySQL Server

## Getting Started

1. Clone the repository:
   ```bash
   git clone <your-repository-url>
   ```
2. Configure your MySQL database using the provided scripts.
3. Build the project:
   ```bash
   mvn clean install
   ```
4. Run the application:
   ```bash
   mvn exec:java
   ```

## Project Structure

- `src/`: Core application source code.
- `tests/`: Unit and integration tests.
- `lib/`: Project dependencies.
- `UML_and_Database_Diagrams.md`: Documentation for system design.
